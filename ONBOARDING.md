# Onboarding — nový počítač nebo nový vývojář

## 1. Naklonuj všechny tři repozitáře do stejné složky

```bash
git clone https://github.com/petrvitek70-pKv/Mastermind
git clone https://github.com/petrvitek70-pKv/MastermindAndroid
git clone https://github.com/petrvitek70-pKv/colegio.cz
```

## 2. Přečti STATUS.md

Obsahuje aktuální stav projektu, co je live, co čeká, co zbývá udělat.

## 3. Zahájení session s Claude

Na začátku každé session Claudovi napiš:

> "Projekty jsou v `/cesta/kde/jsi/klonoval/`. Udělej git pull ve všech třech repozitářích a přečti STATUS.md."

Claude stáhne aktuální změny z GitHubu, přečte STATUS.md a bude okamžitě v obraze — bez nutnosti dlouhého vysvětlování.
